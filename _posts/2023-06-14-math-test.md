Making MathJax work in Github pages is a little annoying, but not too hard. If using a custom layout, skip past the remainder of this sentence, otherwise, copy the minima `_includes` folder to your project's root (use `bundle show minima` to reveal the location). In the `_layouts/default.html` file, inside the `<head/>`, add:

```html
<script type="text/javascript" id="MathJax-script" async
  src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js">
</script>
<script>
  MathJax = {
    tex: {
      inlineMath: [['$', '$']]
    }
  };
</script>
```

Then, `$x + y$` will render as $x + y$ and 

```$$e^{\pi i}=-1$$```

will render as

$$e^{\pi i}=-1$$
